# Recipe Generator 🍳

A simple Python terminal app that suggests meals based on the ingredients, cooking time, dietary preference, and kitchen supplies you enter.

---

## 📌 Features

- 5 built-in meal recipes
- Ingredient matching
- Dietary-preference check
- Cooking-time check
- Kitchen-supply check
- Recipe instructions displayed when a match is found

---

## 🧠 How It Works

1. Enter your available ingredients
2. Enter any dietary preference
3. Enter how much time you have to cook
4. Enter your kitchen supplies
5. The program checks which recipes you can make
6. A matching recipe and its instructions are displayed

---

## ▶️ How to Run

Open a terminal in this folder and run:

```bash
python recipe_generator.py
