# ✊✋✌️ Rock Paper Scissors

A simple Python terminal game where you play Rock Paper Scissors against the computer. First to win rounds builds up a score, and you can keep playing as long as you want.

---

## 🚀 Features

- Classic Rock, Paper, Scissors gameplay
- Computer randomly selects its move
- Score tracking for both player and computer
- Input validation (prevents invalid choices)
- Play multiple rounds in one session
- Final score summary at the end
