# Python Basics

A set of simple Python programs built while learning core programming concepts.

## Projects

### Number Guessing Game

A program where the user tries to guess a randomly generated number.  
The program provides feedback if the guess is too high or too low.

### Rock Paper Scissors

A basic implementation of Rock Paper Scissors where the user plays against the computer.

## Concepts Used

- Variables
- Functions
- Conditionals
- Loops
- User input
- Random number generation

## Notes

These projects are part of my early Python practice and are intended to build foundational skills.
